This is a user requested plug-in. with this plug-in we can setup the RSS slider in our website. in the slider we can configure the title and excerpt. In plug-in administration page, we have option to manage RSS link. maximum we can enter four RSS link in the administration page. we can use widget option or short code option to display the slider in the front end.

**Features of this plugin**

1. Easy installation and customization.
2. Widget option for sidebar.
3. Short code option for posts and pages.
